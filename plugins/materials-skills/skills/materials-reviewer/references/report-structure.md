# Report Structure

Use this structure for each independent report.

## Overall assessment

State Accept, Minor Revision, Major Revision, or Reject.

## Scores

List Innovation and contribution, Methodology soundness, Evidence completeness, Writing quality, Figure/table quality, and Journal fit.

## Major comments

Major comments should be numbered and include severity, location if known, problem, evidence needed, and revision path.

## Minor comments

Minor comments cover wording, labels, missing definitions, units, formatting, and local clarity issues.

## Recommendation

Connect the recommendation to the highest-risk evidence gaps. If the recommendation is Major Revision or Reject, name the minimum evidence needed to change that decision.
