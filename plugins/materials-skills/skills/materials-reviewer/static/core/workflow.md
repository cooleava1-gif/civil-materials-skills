# Reviewer Workflow

1. First read manuscript title, abstract, figures, methods, and conclusions.
2. Detect target journal, material domain, and review depth.
3. Build two independent review perspectives:
   - Reviewer A emphasizes novelty, method rigor, and journal fit.
   - Reviewer B emphasizes mechanism evidence, figures, statistics, and reproducibility.
4. Write each independent review without copying comments between reviewers.
5. Produce the cross-review synthesis after both reviews.
6. End with the combined recommendation and highest-priority revision actions.

The cross-review synthesis should identify agreed issues, disagreed issues, and what must be fixed before resubmission.
