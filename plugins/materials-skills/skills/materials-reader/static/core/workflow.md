# Reading Workflow

1. Identify paper type: experimental, review, methods, field/case, modelling, or mixed.
2. Identify material family: asphalt/pavement, cement/concrete, construction material, sustainability/durability, or generic civil.
3. Build the experiment matrix before writing conclusions.
4. Map each figure/table to the claim it supports.
5. Extract mechanisms only after evidence mapping.
6. Add "borrowable ideas" for the user's project.
7. End with questions to ask in a group meeting or literature review.

For Chinese output, keep technical terms consistent and include English terms in parentheses for important concepts.
