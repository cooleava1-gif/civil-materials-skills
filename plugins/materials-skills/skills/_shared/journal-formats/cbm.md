# Construction and Building Materials (CBM) — Formatting Facts

Authoritative facts about CBM formatting requirements. This file holds the **facts**; each skill's `static/fragments/journal/cbm.md` adds the **action layer**.

> Always verify against the journal's current Guide for Authors before final submission. The numbers below are accurate as of early 2026.
> Official page: https://www.sciencedirect.com/journal/construction-and-building-materials/publish/guide-for-authors

## Article types and limits

| Article type | Word limit | Abstract | References | Figures/Tables | Methods |
|---|---|---|---|---|---|
| **Research article** | No strict word limit (typically 6,000–10,000) | Structured: Background, Methods, Results, Conclusions (~250 words) | No hard cap (~50–80 typical) | No hard cap | Within main text |
| **Review article** | Longer (typically 8,000–15,000) | Unstructured (~300 words) | Comprehensive (100+) | Flexible | N/A |
| **Short communication** | ~3,000 | Structured (~150 words) | ~20 | ~4 | Within main text |

## Abstract

- Structured format: Background, Methods, Results, Conclusions
- ~250 words maximum for research articles
- No citations in abstract
- Include quantitative results where possible

## Figures and tables

| Parameter | Requirement |
|---|---|
| Resolution (halftone/photo) | 300 dpi minimum |
| Resolution (line art) | 1,000 dpi minimum |
| Resolution (combination) | 600 dpi minimum |
| File formats | TIFF, EPS, PDF, or high-quality JPEG |
| Color mode | Color online; grayscale or color in print |
| In-figure font | Legible at publication size |
| Panel labels | Lowercase letters in parentheses: (a), (b), (c) |

## References

- **Style**: Numbered, sequential by first appearance
- **In-text**: Square brackets [1], [2], [1,2], [1–5]
- **Format example**:
  ```
  [1] A.B. Smith, C.D. Johnson, Title of article, J. Constr. Build. Mater. 350 (2025) 123456.
  ```
- Journal names abbreviated per ISO 4
- Include DOI when available

## Mandatory statements

- **Data Availability statement** — required (Elsevier policy)
- **CRediT author statement** — required (individual author contributions)
- **Declaration of competing interest** — required
- **Acknowledgments** — optional but common
- **Ethics statement** — if human/animal subjects involved

## Cover letter

Editors use the cover letter for triage. State explicitly:

1. What the finding is (one sentence)
2. What is novel about it (one sentence)
3. Why it matters to the construction materials community (one sentence)

## Common desk-rejection patterns

1. **No clear construction-material relevance** — pure chemistry or pavement design without material contribution
2. **Single-property study** — only one test type without mechanism or durability context
3. **Missing control group** — improvement claimed without baseline comparison
4. **Weak mechanism evidence** — performance changes attributed to mechanism without spectral/microscopy support
5. **Unsolicited review** — reviews require invitation or editor pre-approval
