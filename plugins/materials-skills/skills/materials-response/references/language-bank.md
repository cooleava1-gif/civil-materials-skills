# Language Bank

Useful response stems:

- We sincerely appreciate the reviewer's insightful comment.
- We agree that the original wording did not sufficiently clarify this point.
- To address this concern, we have revised Section [X] to explicitly state [change].
- The corresponding revision can be found on page [X], lines [X-X].
- We have toned down this claim to avoid overinterpretation.
- Because [reason], this issue is now discussed as a limitation rather than a confirmed mechanism.

Use restrained hedging:

- "suggests" for inferred mechanisms,
- "indicates" for direct measured trends,
- "demonstrates" only for directly supported conclusions.
