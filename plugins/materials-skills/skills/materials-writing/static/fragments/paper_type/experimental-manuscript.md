# Experimental Manuscript

Use this for waterborne epoxy modified emulsified asphalt, cement/concrete, durability, and other data-driven papers.

Core chain:

1. Material problem: current material has a performance or durability limitation.
2. Design rationale: modifier, dosage, or process is selected for a clear reason.
3. test matrix: control, modified groups, curing/conditioning, and mechanism tests.
4. Performance result: measured property improves under stated conditions.
5. Mechanism evidence: FTIR, SEM, fluorescence, rheology, XRD, TG/DTG, or other relevant evidence explains the trend.
6. Boundary: field durability or broad mechanism claims require additional validation.

For waterborne epoxy modified emulsified asphalt, connect bonding performance to demulsification, curing, storage stability, viscosity, moisture conditioning, and interface morphology.
