# Methods

Methods must make the experiment reproducible.

Include:

- binder/emulsion source and specification,
- epoxy resin and curing agent information,
- dosage definition,
- mixing speed, duration, and temperature,
- curing age,
- substrate or aggregate condition,
- test standard or adapted protocol,
- replicate count and statistics.

If a method detail is unknown, keep a placeholder instead of smoothing over it.
