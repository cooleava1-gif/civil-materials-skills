# Polishing Workflow

1. Identify section and journal family.
2. Identify claim strength: measured, inferred, speculative, or practical implication.
3. Tighten sentence structure.
4. Replace promotional language with evidence-weighted language.
5. Improve transitions and paragraph logic.
6. Check terminology consistency.
7. Return polished text and a compact claim-risk note.

Default sentence target: 15-28 words. Longer sentences are acceptable only when technical structure requires it.
