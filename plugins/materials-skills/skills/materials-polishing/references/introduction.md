# Introduction Polishing

The introduction should move from:

engineering problem -> material strategy -> unresolved bottleneck -> study objective -> evidence plan.

Civil materials introductions should not read like a polymer chemistry review unless chemistry is the central contribution.

## Before/After Examples

### Example 1: From literature list to gap

Before:

> Many researchers have studied emulsified asphalt, waterborne epoxy, polyurethane, SBR latex, and other modifiers. These studies show good results.

After:

> Polymer modification has been used to improve the cohesion and adhesion of emulsified asphalt systems, yet the balance between emulsion stability, epoxy curing, viscosity growth, and interface bonding remains insufficiently resolved.

Why: the revision turns a list of materials into a research bottleneck that can motivate the study.

### Example 2: Narrowing chemistry detail

Before:

> Epoxy resin contains epoxy groups, and amine curing agents react with epoxy groups, so the reaction mechanism is very complex and important.

After:

> For waterborne epoxy-modified emulsified asphalt, the relevant question is not only whether epoxy curing occurs, but whether curing can develop without compromising demulsification, workability, and interlayer adhesion.

Why: the revision keeps chemistry only where it supports the materials problem.
