# Example: Figure Package

Module: materials-figure

## Use Case

Create a CBM-style figure package for bonding strength and mechanism evidence.

## Example Output Shape

- Figure intent: compare control and modified groups.
- Data structure: sample, dosage, mean, SD, n, standard.
- Caption: claim -> evidence -> boundary.
- Reviewer risk: performance does not prove mechanism.

## Quality Bar

Every figure needs data source, unit, condition, caption boundary, and missing-data flags.
