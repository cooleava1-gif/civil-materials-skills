# Example: PPTX Outline JSON

Module: materials-pptx

## Use Case

Prepare a real PPTX deck from a verified outline.

## Example Output Shape

```json
{
  "title": "Waterborne Epoxy Modified Emulsified Asphalt",
  "slides": [
    {"layout": "title", "title": "Research Background"},
    {"layout": "figure", "title": "Bonding Strength", "evidence_status": "data needed"}

```

## Quality Bar

Do not fabricate values or images. Use placeholders when data are missing.
