# Task: Figures and Tables

Goal: make figures/tables that support claims, not decoration.

For actual journal-ready figure planning, SVG chart generation, or figure-package audits, prefer the companion skill `materials-figure`.

Before plotting, write:

- Main conclusion of the figure.
- Data groups and controls.
- Statistical display plan.
- Journal/export requirement if known.

Common materials figure set:

- Materials and experimental workflow schematic.
- Dosage-response performance plot.
- Durability retention plot.
- Microstructure or chemical evidence panel.
- Mechanism schematic.
- Summary radar or decision matrix only if it clarifies trade-offs.

Tables should carry dense comparison:

- Literature comparison table for reviews.
- Mix design or formulation table.
- Test standards and conditions table.
- Reviewer-risk checklist table.

Avoid overloaded multi-panel figures where every panel says "higher is better".
