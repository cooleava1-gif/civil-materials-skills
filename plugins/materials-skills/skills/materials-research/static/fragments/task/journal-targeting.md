# Task: Journal Targeting

Goal: match manuscript contribution to journal fit and reviewer expectations.

Use a decision table with:

- Journal.
- Best fit.
- Weak fit.
- Required evidence.
- Likely reviewer concerns.
- Submission action.

Do not rely on memory for:

- current impact factor,
- APC,
- open access status,
- indexing,
- article type restrictions,
- exact formatting rules.

Verify official journal pages for current submission advice.

Common routing:

- Broad construction material experiments -> CBM or JBE.
- Road/asphalt/pavement materials -> CBM in Transportation, RMPD, IJPE, sometimes CBM.
- Cement/concrete composites with mechanism depth -> CCC.
- Applied case studies or locally sourced construction materials -> CSCM.
- Review manuscripts -> verify whether the target accepts unsolicited reviews before recommending.
