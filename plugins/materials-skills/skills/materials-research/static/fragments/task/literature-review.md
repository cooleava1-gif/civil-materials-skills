# Task: Literature Review

Goal: build a review that explains a field, not a list of papers.

Recommended structure:

1. Engineering problem and service environment.
2. Material modification strategies.
3. Performance evaluation methods.
4. Mechanism evidence and unresolved debates.
5. Durability and constructability constraints.
6. Standardization, field application, and future research.

For a short review, prioritize:

- one focused material system,
- one core performance problem,
- a table of methods/results/mechanisms,
- a final gap map.

Avoid:

- paper-by-paper chronological summaries,
- unsupported "research is still insufficient" endings,
- broad claims about sustainability without boundary definitions.

For road/asphalt topics, include a table separating binder/emulsion properties, mixture or interface properties, and engineering-service properties.
