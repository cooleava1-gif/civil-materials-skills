# Task: Data Analysis and Result Interpretation

Goal: convert raw or summarized data into defensible materials claims, figures, and manuscript paragraphs.

Required workflow:

1. Identify variables, controls, units, specimen count, and test condition.
2. Separate descriptive trends from statistically supported claims.
3. Map each trend to a figure/table.
4. Link each figure/table to one manuscript claim.
5. Flag missing replication, variability, or service-condition data.

Default output:

- Main trend.
- Best figure/table type.
- Claim sentence.
- Mechanism interpretation.
- Missing evidence or caution.

For common materials data:

- Dosage-response data -> line plot or grouped bars with optimum discussion.
- Retained strength after water/ageing -> retention ratio plot.
- Storage stability or viscosity -> constructability trade-off figure.
- Microstructure images -> qualitative panel plus quantified support if available.

Do not convert a weak trend into a strong mechanism claim.
