# Task: PPTX Generation

Goal: create an actual PowerPoint `.pptx` file.

Use `materials-pptx` for the file artifact.

Use `materials-paper2ppt` first if the slide story, audience, or section order is not yet clear.

Before handoff, identify:

- audience,
- language,
- deck type,
- source outline or manuscript,
- whether figures/data are ready.
