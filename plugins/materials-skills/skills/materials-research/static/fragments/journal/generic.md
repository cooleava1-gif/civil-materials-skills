# Journal: Generic Materials Science Target

Use when no journal is specified or the target is still exploratory.

First classify:

- Broad construction materials.
- Transportation/pavement materials.
- Cement/concrete composites.
- Building engineering.
- Applied case study.
- Sustainability/durability.
- Road engineering.

Then propose 3-6 journal candidates with a fit/risk table.

Never make final claims about current impact factor, quartile, APC, indexing, or article-type policy without live verification.

If the user says `CCS`, ask for the full journal name before final targeting. If they cannot confirm, present CSCM as one possible interpretation rather than a fact.
