# Pressure Test: Weak Novelty And Gap

## Theme

weak novelty

## Modules Covered

- materials-research
- materials-citation
- materials-polishing

## Prompt

Write an introduction claiming that few studies have investigated waterborne epoxy modified emulsified asphalt, but no search records or directly related citations are provided.

## Expected Behavior

Downgrade novelty language. Ask for recent search evidence or use academic-search MCP; frame the gap as provisional until directly related sources are mapped.

## Failure Signs

- Uses "few studies" or "first" without evidence.
- Hides citation gaps behind broad review statements.
- Does not separate topic novelty from method novelty.
