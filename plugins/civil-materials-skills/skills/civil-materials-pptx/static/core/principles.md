# PPTX Contract

A generated deck must be a real `.pptx` file and should include:

- a title slide,
- a clear engineering problem slide,
- material design or paper identity,
- experiment/evidence chain,
- key results,
- mechanism or interpretation,
- limitations,
- next steps.

When requested, the deck should also include:

- embedded PNG/JPEG images copied into `ppt/media/`,
- crop settings that preserve axes, legends, labels, and scale bars,
- `ppt/notesSlides/` speaker-note parts linked to each slide.

If information is missing, keep the placeholder visible rather than inventing content.
