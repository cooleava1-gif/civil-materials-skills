# Polishing Contract

Preserve:

- data,
- units,
- test conditions,
- citation intent,
- uncertainty,
- limitations,
- author meaning.

Improve:

- sentence clarity,
- paragraph flow,
- evidence-weighted verbs,
- journal-appropriate tone,
- terminology consistency.

Flag rather than silently fix:

- unsupported novelty,
- missing statistical evidence,
- vague mechanism claims,
- sustainability claims without boundary,
- "significant" without statistical support.

## Paper Production Handoff

When polishing routed paper-production text, use the incoming `evidence_level`
and `weakness route` to control claim strength. If a weakness row says the
evidence is performance-only, speculative, or missing, downgrade causal wording
and record a reviewer-safe `claim strength` note instead of making the prose
sound more certain.
