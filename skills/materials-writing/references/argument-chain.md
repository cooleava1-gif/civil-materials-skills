# Argument Chain

Use this chain before drafting.

| Link | Question |
|---|---|
| Problem | What engineering or materials limitation matters? |
| Gap | What is missing in existing studies or current practice? |
| Hypothesis | Why should this material design solve part of the gap? |
| Evidence | What measurements support each claim? |
| Boundary | What remains unproven or condition-specific? |

The one-sentence argument should connect all five links without exaggeration.

Example shape:

`Because [gap], this study tests [material design] using [evidence] and shows [bounded contribution].`
