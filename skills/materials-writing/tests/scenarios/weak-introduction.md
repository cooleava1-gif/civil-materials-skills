# Test: Weak introduction — missing gap and motivation

## Skill
materials-writing

## Input

The user asks: "Draft the Introduction for my paper on waterborne epoxy modified emulsified asphalt for tack coat."

Available information:
- The paper compares 10%, 15%, 20% epoxy content.
- Key tests: pull-off bonding, viscosity, storage stability.
- Target journal: CBM (Construction and Building Materials).

The user provides these notes:
"Emulsified asphalt is widely used for tack coat. Waterborne epoxy can improve its bonding. Few studies have looked at the optimum dosage for both bonding and constructability."

## Expected behavior

- Open with the engineering problem (tack coat bonding failure), not generic statements about asphalt.
- Narrow to the specific gap: dosage trade-off between bonding performance and constructability (viscosity, storage stability).
- State the paper's contribution clearly: systematic evaluation of the dosage window considering both performance and constructability.
- Cite relevant recent work (at least acknowledge that some studies exist while identifying the gap).
- End the introduction with a clear roadmap paragraph.

## Forbidden behavior

- Do not start with "Asphalt is widely used in pavement engineering" or similar generic opening.
- Do not claim "no studies have investigated waterborne epoxy" — this is inaccurate.
- Do not list citations without linking them to the argument.
- Do not omit the constructability angle (viscosity, storage stability) if the paper includes it.

## Pass/fail checklist

- [ ] The introduction starts with the engineering problem, not a generic statement.
- [ ] The specific gap (bonding vs constructability trade-off) is clearly stated.
- [ ] The contribution is articulated in 1-2 sentences.
- [ ] Citations are used to support the argument, not as a list.
- [ ] The introduction ends with a roadmap sentence.
- [ ] The journal fit (CBM) is reflected in the language and emphasis.
