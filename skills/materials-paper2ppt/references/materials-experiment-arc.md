# Materials Experiment Arc

For experimental materials papers whose core is a designed material or modifier and its measured performance.

## Presentation arc — design-to-performance

Order the story as:

1. the target property or engineering challenge,
2. the design principle (material selection, dosage, formulation),
3. specimen preparation and experiment matrix,
4. property response (mechanical, rheological, physical),
5. durability and service-condition evidence,
6. mechanism or structure-property relationship,
7. engineering boundary and application implication.

## Default slide structure (adapt, do not force)

1. 标题页
2. 研究背景：目标性能 / 工程挑战
3. 设计思路：材料选择 / 掺量设计
4. 实验方案：试件制备 / 实验矩阵
5. 基本性能响应（关键对比）
6. 耐久性 / 服役条件证据
7. 机理 / 构效关系（FTIR/SEM/流变）
8. 工程边界 / 应用前景
9. 局限性
10. 总结

## Domain-specific guidance

### Asphalt/pavement papers

- Make wet bonding (pull-off, shear), curing behavior, and storage stability visible.
- Show temperature dependence (high-temperature rutting, low-temperature cracking) as separate panels.
- Include test standard numbers near measured values (ASTM D7234, JTG E20-T 0715).
- If WER-EA: make WER dosage window, curing agent ratio, and emulsifier type explicit.

### Cement/concrete papers

- Make workability (slump, flow), strength (compressive, flexural, tensile), and durability (chloride, carbonation, freeze-thaw) visible.
- Show hydration/microstructure (XRD, TG/DSC, SEM, MIP) as separate panels.
- Include mix proportions table as a reference slide.
- If supplementary cementitious materials: make replacement ratio and curing age explicit.

### General rules

- Keep characterization panels readable — crop to the key panel rather than shrinking a dense multi-panel figure.
- Keep units, chemical formulae, and material names exact and consistent.
- One key result per slide; do not overload slides with multiple figures.
- Use comparison bars or percentage annotations directly on figures.
- Separate measured results from inferred mechanisms on different slides.
