# Task: Reviewer Response

Goal: turn reviewer comments into respectful, evidence-bound point-by-point replies.

Use `materials-response` for production work.

Before handoff, identify:

- target journal,
- reviewer comment list,
- concern type,
- manuscript actions needed,
- available evidence versus missing evidence.

If a requested experiment or analysis is not available, mark it as `[needs author decision]` rather than pretending it has been done.
