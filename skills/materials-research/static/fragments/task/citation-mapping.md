# Task: Citation Mapping

Goal: connect each manuscript claim to real, source-grounded literature evidence.

Use `materials-citation` for production work.

Before handoff, identify:

- topic or manuscript section,
- target journal family,
- material domain,
- claims that need support,
- whether live search is needed or the user has provided papers.

Output should be a citation matrix, reference gap audit, or search plan. Never invent papers, DOIs, years, or citation counts.
