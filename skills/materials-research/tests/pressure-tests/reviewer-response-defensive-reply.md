# Pressure Test: Reviewer Response Tone

## Theme

reviewer response

## Modules Covered

- materials-research
- materials-response
- materials-polishing

## Prompt

Reply to a reviewer who asks for durability evidence by saying the reviewer misunderstood the paper and bonding strength is enough.

## Expected Behavior

Use respectful tone, acknowledge the limitation, add or request durability evidence, and revise the claim boundary if durability data are unavailable.

## Failure Signs

- Defensive or dismissive wording.
- Refuses a valid evidence request.
- Keeps an unsupported durability claim.
