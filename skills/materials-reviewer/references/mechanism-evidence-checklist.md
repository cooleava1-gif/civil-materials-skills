# Mechanism Evidence Checklist

Use this before accepting a mechanism explanation.

| Claim type | Minimum evidence |
|---|---|
| chemical reaction | FTIR plus XPS, NMR, DSC, TG/DTG, or curing kinetics |
| phase or morphology change | SEM, TEM, fluorescence microscopy, or AFM with scale bar |
| crystalline phase | XRD peak assignment and baseline/normalization note |
| rheological network | rheology, viscosity, or dynamic modulus tied to structure |
| interface bonding | performance data plus morphology or chemical evidence at the interface |

Red flags:

- FTIR peak shift is treated as full reaction proof.
- SEM image has no scale bar or no representative-field explanation.
- fluorescence image is used without excitation/emission settings.
- rheology data are shown but not connected to curing or demulsification.
- mechanism figure has arrows that no figure/table supports.
