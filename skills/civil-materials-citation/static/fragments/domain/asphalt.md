# Asphalt Domain Citation Rules

For asphalt and pavement manuscripts, keep evidence separated by these layers:

- **Binder/Emulsion**: base asphalt type, emulsifier type and dosage, modifier type and dosage, basic properties.
- **Interface/Bonding**: tack coat materials, bonding strength test method, interface treatment, curing condition.
- **Mixture**: aggregate gradation, mixture design, volumetric properties, mechanical performance.
- **Construction**: construction temperature, curing time, traffic opening condition.
- **Service-condition**: moisture damage, aging, freeze-thaw, fatigue, rutting.

Separate mechanism citations from performance citations.
