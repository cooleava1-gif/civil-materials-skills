# Example: Waterborne Epoxy Emulsified Asphalt FAIR Package

## Scenario

Topic: waterborne epoxy modified emulsified asphalt bonding performance.

Target journal: CBM, RMPD, IJPE, or JBE.

## Minimum Package

- `raw_data/experiment_data_template.csv`
- `processed_data/`
- `figures/`
- `metadata.md`
- `README.md`
- `data_availability_statement.md`
- `fair_audit.md`

## Required Metadata

- asphalt_type
- emulsifier_type
- waterborne_epoxy_type
- epoxy_dosage
- curing_condition
- demulsification_condition
- test_standard
- temperature
- humidity
- aging_condition
- replicate_count

## Reviewer Risk

If bonding strength is present but FTIR, SEM, fluorescence microscopy, or rheology metadata are missing, do not claim a curing or interface mechanism.
