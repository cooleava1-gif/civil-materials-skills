# Workflow

1. Convert the manuscript topic into 3-6 searchable concept blocks.
2. Convert claims into evidence needs.
3. Build search strings for broad discovery and narrow confirmation.
4. Screen sources by relevance, evidence type, journal fit, and recency.
5. Map each selected source to the exact manuscript claim it supports.
6. Flag unsupported claims, over-broad claims, and citation stacking.

Prefer a compact matrix over a prose-only literature summary.
