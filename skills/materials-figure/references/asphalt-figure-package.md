# Asphalt Figure Package

Recommended package for waterborne epoxy modified emulsified asphalt:

1. formulation and workflow schematic,
2. viscosity/storage stability versus epoxy dosage,
3. dry/wet bonding strength and retained ratio,
4. failure mode images,
5. FTIR or morphology evidence,
6. mechanism schematic.

Caption stance: dosage optimum and trade-off are more important than "higher is better".
