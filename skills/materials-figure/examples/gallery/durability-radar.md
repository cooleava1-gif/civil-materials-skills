# Gallery Card: Durability Radar

## Figure Intent

Summarize retained performance across moisture damage, aging, freeze-thaw, fatigue, rutting, or service-condition indicators.

## Data Structure

| metric | control_normalized | modified_normalized | raw_unit | test_condition |
|---|---:|---:|---|---|
| Moisture retention |  |  | % |  |
| Aging retention |  |  | % |  |

## Caption Pattern

Normalized durability profile of `[material system]`. Values are normalized against `[reference]`; larger values indicate `[better retained performance/lower damage]`.

## Reviewer Risk

Radar charts can hide raw values and uncertainty. Include raw data in a table or supplementary figure when used for manuscript claims.

## Borrowing Note

Use this card for review summaries, thesis reports, and multi-property trade-off figures.
