# Gallery Card: Mechanism Schematic

## Figure Intent

Explain a proposed material mechanism, such as demulsification, epoxy curing, asphalt-resin compatibility, interface bonding, and moisture resistance.

## Data Structure

| mechanism_step | supporting_evidence | evidence_status | manuscript_boundary |
|---|---|---|---|
| Demulsification |  | measured/inferred/missing |  |
| Epoxy curing |  | measured/inferred/missing |  |

## Caption Pattern

Schematic illustration of the proposed `[material system]` mechanism. Solid arrows indicate measured evidence; dashed arrows indicate inferred links requiring further verification.

## Reviewer Risk

A schematic is explanatory, not direct proof. It must cite or point to FTIR, SEM, rheology, bonding, durability, or other measured evidence.

## Borrowing Note

Use this card for graphical abstracts, review mechanism summaries, and PPT explanation slides.
