# Gallery Card: Bonding Strength Bar

## Figure Intent

Compare the bonding strength of control emulsified asphalt and modified systems, such as waterborne epoxy modified emulsified asphalt at different dosages.

## Data Structure

| group | dosage | mean_strength | standard_deviation | n | test_standard |
|---|---:|---:|---:|---:|---|
| Control | 0% |  |  |  |  |
| WE-5 | 5% |  |  |  |  |

## Caption Pattern

Bonding strength of emulsified asphalt systems under `[test condition]`. Bars represent mean values and error bars represent `[SD/SE]` from `n = [replicates]`.

## Reviewer Risk

This figure proves a measured performance difference only. It does not prove chemical curing, interface morphology, or moisture durability unless those measurements are shown elsewhere.

## Borrowing Note

Use this card for CBM/JBE/RMPD performance figures and group-meeting slides that need a clean control-vs-modified comparison.
