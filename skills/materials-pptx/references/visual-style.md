# Visual Style

Use a clean engineering style:

- wide 16:9 format,
- short titles,
- 3-5 bullets per slide,
- high contrast,
- restrained accent colors,
- figures larger than explanatory text,
- mechanism arrows only where evidence supports causality.

Recommended color roles:

- asphalt/pavement: deep slate plus amber accent,
- cement/concrete: warm gray plus blue accent,
- sustainability: muted green plus graphite.
