# Test Standards Mapping

Use this reference to keep civil-materials claims tied to recognized test methods. Always verify the latest official standard text before final submission.

> Companion reference: [standards-mapping.md](standards-mapping.md) provides GB/T, JTG, ASTM, EN, and ISO mapping by material class with manuscript wording templates and citation cautions. Use both references for complete coverage.

## Asphalt and Pavement Materials

| Evidence need | Common standards or families | Report with the result |
|---|---|---|
| Asphalt binder penetration, softening point, ductility | JTG E20, ASTM D5, ASTM D36, ASTM D113 | binder type, temperature, aging state |
| Emulsified asphalt residue and stability | JTG E20 emulsion tests, ASTM D244 | emulsion type, residue method, storage time |
| Viscosity of asphalt emulsion or binder | ASTM D2196, ASTM D4402, JTG E20 | temperature, spindle/geometry, shear rate |
| Tack coat bond or interface shear | Project/lab protocol, pull-off/shear fixtures, AASHTO/ASTM-adapted methods where applicable | substrate, application rate, curing condition, loading rate |
| Moisture damage and freeze-thaw durability | AASHTO T283, EN/JTG adapted freeze-thaw methods | conditioning cycle, retained strength ratio |
| Asphalt mixture rutting/fatigue | JTG E20, AASHTO, EN 12697 family | specimen size, air voids, temperature, loading protocol |

## Cement, Concrete, Mortar, and SCMs

| Evidence need | Common standards or families | Report with the result |
|---|---|---|
| Workability/slump/flow | ASTM C143, ASTM C1437, EN 12350 | mixing sequence, elapsed time, temperature |
| Compressive strength | ASTM C39, ASTM C109, EN 12390 | specimen size, curing age, curing condition |
| Flexural/splitting tensile strength | ASTM C78, ASTM C496, EN 12390 | loading rate, specimen geometry |
| Setting time | ASTM C191, ASTM C403 | paste/mortar system, temperature |
| Chloride penetration/resistivity | ASTM C1202, ASTM C1556, AASHTO T358 | saturation condition, age, voltage/current protocol |
| Drying shrinkage | ASTM C157 | curing duration, humidity, specimen size |
| Microstructure and hydration | XRD, FTIR, TG/DTG, SEM, MIP protocols | sample preparation, drying/solvent exchange, peak assignment basis |

## Durability and Sustainability

| Evidence need | Common standards or frameworks | Boundary note |
|---|---|---|
| Freeze-thaw resistance | ASTM C666, EN 12390, JTG/GB methods | Do not generalize to field service without exposure context. |
| Carbonation/chloride/sulfate attack | RILEM, ASTM, EN, GB/T method families | State solution concentration, exposure age, and specimen conditioning. |
| Life-cycle or carbon claims | ISO 14040/14044, EN 15804, journal-specific LCA rules | Define functional unit, system boundary, allocation, and data source. |
| Recycling and circularity | JCP/RCR-style inventory reporting | Do not write `green` without quantified impact category evidence. |

## Reviewer-Safe Usage

- Cite the exact standard number and year in the manuscript when possible.
- If a lab method is adapted, state what was adapted and why.
- If no standard exists for a custom interface test, define the fixture, loading rate, curing state, temperature, and failure criterion.
- Do not compare results across papers unless the specimen preparation and test standard are compatible.
