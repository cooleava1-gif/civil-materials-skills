# Workflow

1. Number every reviewer comment.
2. Classify the concern: novelty, method, mechanism, data, statistics, writing, scope, figure/table, citation, or ethics.
3. Choose a response strategy: accept and revise, clarify and revise, rebut with evidence, add limitation, or defer with justification.
4. Link each response to a manuscript action.
5. Audit whether the action is verifiable in the revised text.
6. Polish tone after the technical response is correct.
