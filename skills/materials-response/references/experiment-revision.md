# Experiment Revision

When reviewers request more evidence, separate what can be added from what must be framed as a limitation.

| Concern | Strong revision | If data are unavailable |
|---|---|---|
| Mechanism unclear | add FTIR/SEM/fluorescence/rheology evidence and revise mechanism diagram | state mechanism as hypothesis and soften claim |
| Durability insufficient | add moisture, aging, freeze-thaw, or retained strength evidence | narrow conclusions to short-term performance |
| Statistics missing | add replicate count, error bars, significance or variability discussion | mark as preliminary and avoid overclaiming |
| Comparison weak | add baseline/control or literature benchmark | justify control choice and add limitation |

Never invent completed experiments. Use `[additional experiment needed]` until the user confirms data.
