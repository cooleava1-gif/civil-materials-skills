# Example: RMPD/IJPE Minor Revision Response

## Reviewer Comment

The manuscript is generally suitable for pavement materials readers, but the authors should clarify the tack coat application rate, test temperature, and whether the reported bond strength values are averages of repeated specimens.

## Good Response

We have clarified these experimental details in the revised manuscript. The tack coat application rate, curing time, test temperature, and replicate count are now reported in the Methods section and repeated in the relevant figure caption so that the bond-strength results can be interpreted within their pavement-service context.

Revision made:

- Methods: added application rate, curing time, test temperature, substrate condition, loading rate, and replicate count.
- Figure caption: added `mean value of n = [x] specimens; error bars represent [SD/SE/CI]`.
- Discussion: added one sentence limiting the conclusion to the tested application rate and temperature.
- Manuscript location: [insert page/line after revision].

## Why This Works

- It answers a minor revision without becoming defensive.
- It uses pavement-language details that RMPD/IJPE reviewers expect.
- It avoids overgeneralizing bond strength beyond the tested condition.
- It separates method clarification from performance interpretation.
